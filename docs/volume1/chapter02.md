# 第2話　図書館での再会

　新学期三日目の放課後。僕は図書館の奥の席で、『アルゴリズムとデータ構造』という分厚い本と格闘していた。

　今日読んでいるのは、線形時間選択アルゴリズムの章だ。配列のn個の要素からk番目に小さい要素を見つける問題――一般的にはO(n log n)の時間がかかるソートを使うところを、クイックセレクトを改良したアルゴリズムでO(n)まで短縮できる。

　理論上は美しいアルゴリズムだけれど、実装となると話は別だ。メディアンオブメディアンズ法の詳細を追いかけていると、頭の中で抽象的な概念が具体的なコードに変換されていく感覚がある。

「すみません、数学の参考書はどの辺りにありますか？」

　図書館の静寂を破って、馴染みのある声が響いた。

　僕は反射的に顔を上げる。カウンターで司書の先生と話しているのは――美月だった。

　制服に身を包んだ美月が、少し困ったような表情で案内を受けている。その横顔を見ていると、なぜか胸の奥がざわめいた。

　美月は数学のコーナーに向かいながら、図書館を見回している。そして僕を見つけた瞬間、その表情が明るくなった。

「あ、拓海先輩」

　驚いたような、でも嬉しそうな声で僕の名前を呼ぶ。その声音に含まれた微妙な感情の変化に、僕の心臓が小さく跳ねた。

「美月。勉強？」

「はい。数学の復習をしようと思って」

　美月は僕の斜向かいの席に座りながら、困ったような笑顔を浮かべた。「高校の授業、中学よりもずっと難しくて……」

　距離が近い。僕のノートパソコンの画面に映ったコードが、なぜか急に恥ずかしく感じられた。線形時間選択なんて、美月には全く関係のない世界だ。

「どの辺りが分からない？」

「微分なんです。特に、導関数の定義がよく理解できなくて」

　美月は持参した数学の教科書を開いて見せてくれた。高校一年生にとって微分は確かに難しい。でも美月の場合、単に理解力が足りないわけではない。彼女なりに真剣に向き合おうとしているのがわかる。

「微分っていうのは……」

　僕は手近にあったノートを取り出した。そして、ペンを握る手が微かに震えていることに気づく。美月がすぐ隣にいるからだ。

　ノートに座標軸を描きながら、僕は慎重に言葉を選んだ。

「関数の変化の瞬間的な割合を求める操作なんだ。例えば、この放物線y = x²の上の点(a, a²)での接線の傾きを知りたいとする」

　美月は僕の手元を真剣に見つめている。その集中した表情、少し眉をひそめて考える仕草、時々「ああ」と小さく頷く様子――全てが、いつもの美月なのに、なぜか今日は特別に見えた。

「傾きを求めるには、その点を含む二点を結んだ直線の傾きの極限を考える。つまり……」

　僕は図を描きながら説明を続けた。数学的な美しさと論理の明快さが、ペンの動きとともに紙の上に現れていく。

「(a+h, (a+h)²)との傾きは、((a+h)² - a²)/h = (2ah + h²)/h = 2a + h。これでhを0に近づけると……」

「2aになる！」

　美月が目を輝かせて答えた。その瞬間の彼女の表情は、まるで新しい世界の扉が開かれたような驚きに満ちていた。

「そういうこと。微分係数f'(a) = 2aが得られる」

「すごい……なんだか、数学って生きてるみたいですね」

　美月の感想に、僕は思わず微笑んだ。プログラミングでも同じことを感じる。アルゴリズムが動作する様子、データが変換されていく過程――全てが生きているようだ。

「プログラミングでも同じなんだ。微分は数値計算や機械学習で頻繁に使う」

「プログラミングで？」

「例えば、最急降下法っていうアルゴリズムがあるんだけど、関数の最小値を見つけるために微分を使う。AIが学習する時の基本的な仕組みでもある」

　美月の目がさらに輝いた。

「拓海先輩の世界って、本当にすごいんですね。数学とプログラミングが繋がってて」

　その純粋な尊敬の眼差しに、僕は胸が熱くなった。同時に、プレッシャーも感じる。美月の期待に応えられるだろうか。IOIで金メダルを取れるだろうか。

「美月の理解力が高いからだよ。今の説明だって、すぐに理解してくれた」

「本当ですか？」

「本当だよ。もっと練習してみる？」

　美月は嬉しそうに頷いた。

　それから一時間、僕たちは数学の世界に没頭した。美月は疑問に思ったことは遠慮なく質問してくれるし、理解した時の表情は本当に美しい。教えている僕の方も、いつの間にか夢中になっていた。

「ありがとうございました。本当によくわかりました」

　時計を見ると、もう五時を過ぎていた。美月は深々と頭を下げる。

「そんなに丁寧にお礼を言わなくても」

「でも、拓海先輩のおかげです。私、数学が苦手で」

「美月は苦手じゃない。証拠に、今日習ったことはもう完璧に理解してる」

　僕の言葉に、美月の表情が明るくなった。

「本当に、ありがとうございます。拓海先輩にそう言ってもらえると、頑張れます」

　その言葉に、僕の心が静かに震えた。美月の笑顔のために、もっと勉強を教えてあげたい。彼女の理解が深まっていく過程を、もっと見ていたい。

　そしてその静寂の中で、僕は自分の心の変化を感じ取っていた。美月と過ごす時間が、いつの間にか特別な意味を持ち始めている。それがなんなのか、まだはっきりとはわからないけれど――

「そういえば、新しいクラスはどう？」

　昇降口で靴を履き替えながら、僕は美月に聞いてみた。

「みんな優しくて、楽しいです。友達もできました」

「よかった。心配してたんだ」

「心配……ですか？」

　美月が振り返った。その瞬間、僕の心臓が一拍置いて鼓動した。

「だって、新しい環境だから。馴染めるかなって」

「拓海先輩がそんなに心配してくれてたなんて……」

　美月の声が小さくなった。何か言いたそうな表情をしているけれど、結局何も言わずに歩き続ける。

「幼馴染だから、当然だよ」

「そうですね……幼馴染」

　美月はそう呟いて、少し複雑な表情を見せた。その表情の意味が、僕には読み取れない。でも、何かが変わろうとしている――そんな予感だけは確かに感じられた。

　学校を出ると、夕日が校門の向こうに見えた。桜の花が数輪咲いて、夕日に照らされて綺麗だった。

「桜、少し咲きましたね」

「そうだね。来週には満開になるかな」

「楽しみです。桜の季節って好きなんです」

「どうして？」

「新しいことが始まる感じがするから。それに、なんだかわくわくするんです」

　美月はそう言って桜の花を見上げた。夕日に照らされた横顔が、今日は特に美しく見えた。

「美月」

「はい？」

「今度、一緒にお花見でもしない？」

　その提案に、美月の表情が明るくなった。

「いいんですか？」

「もちろん。桜の季節だし」

「ありがとうございます。ぜひお願いします」

　美月は嬉しそうに手を叩いた。その無邪気な仕草に、僕の胸が温かくなる。

　家まで歩く道すがら、僕たちは他愛ない話をした。美月の新しいクラスメートのこと、プログラミング部のこと、最近読んだ本のこと。なんでもない話だけれど、美月と話していると時間があっという間に過ぎる。

「あ、私の家です」

　気がつくと、美月の家の前に着いていた。桜月堂の看板が、夕日に照らされて温かく光っている。

「お疲れ様でした。明日もよろしくお願いします」

「こちらこそ。お疲れ様」

　美月は門の前で振り返って、もう一度お辞儀をした。そして家の中に入っていく。

　僕は美月の後ろ姿を見送りながら、胸の奥で何かがざわめくのを感じた。これは何だろう。美月が他のクラスメイトと仲良くなって、僕との時間が減ってしまうのではないかという不安？

　いや、それだけじゃない。もっと複雑な、もっと深い何かがある。

　部屋に戻って、今日の出来事を思い返していた。図書館での美月の真剣な表情、微分を理解した時の嬉しそうな顔、夕日に照らされた横顔。どれも綺麗で、印象に残っている。

　デスクの上に置いてある、さっき美月に説明した微分のノートを見る。美月の字で「ありがとうございました」と書かれたメモが挟まっていた。

　胸の奥が温かくなった。これからも、美月の勉強を手伝ってあげよう。そうすれば、もっと一緒にいる時間が増える。

　でも、その理由は果たして「幼馴染だから」なのだろうか。

　窓の外を見ると、美月の家の窓に明かりが灯っていた。きっと今頃、今日習った数学の復習をしているのかもしれない。

　僕も宿題を始めようと思ったけれど、なんだか集中できなかった。頭の中に、美月の笑顔が浮かんでは消える。

　明日も、図書館で一緒に勉強できるだろうか。そんなことを考えながら、僕は机に向かった。しかし開いたページは、さっき読んでいた線形時間選択アルゴリズムではなく、微分の教科書だった。

　美月に教えるために、もっと分かりやすい説明方法を考えよう――そう思いながら、僕は夜が更けるまで数学の本と向き合っていた。
