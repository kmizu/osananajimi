# 第4話　放課後の約束

　月曜日の昼休み、僕は一年生の校舎を訪れていた。廊下には春の午後の陽射しが差し込んで、新入生たちの賑やかな声が響いている。美月のクラスを見つけて、教室の入り口で声をかける。

「美月」

「あ、拓海先輩」

　美月は友達と話していたところだったが、僕を見つけて嬉しそうに立ち上がった。高校生になって少し大人っぽくなった美月を見ると、なぜか胸がざわつく。

「ちょっといい？」

「はい」

　美月は友達に軽く会釈してから、僕の方に歩いてきた。その歩き方一つとっても、中学生の頃より随分と洗練されたように見える。

「体験会の件で話があるんだ」

「廊下で話しましょうか」

　僕たちは教室を出て、人通りの少ない廊下の端に移動した。窓の外では、校庭の桜が三分咲きほどになっている。

「水曜日の体験会、午後四時から六時までの予定なんだ」

「分かりました。文芸部の活動は五時までなので、途中から参加させてもらってもいいですか？」

「もちろん。無理しないで」

「ありがとうございます」

　美月は笑顔で答えてくれた。その笑顔を見ていると、なんだか時が止まったような気がする。

「それから、一つお願いがあるんだけど」

「何でしょうか？」

「もしよかったら、今日の放課後、少し時間もらえる？プログラミングの基本的なことを説明しておきたくて」

　僕がそう言うと、美月は少し驚いたような顔をした。

「私のためにですか？」

「うん。水曜日にいきなり参加してもらうより、少し予習しておいた方がいいかなと思って」

「ありがとうございます。でも、そこまでしてもらって」

「気にしないで。僕も説明の練習になるし」

　実際は、美月ともう少し時間を過ごしたいという気持ちの方が大きかった。でも、それを正直に言うのは恥ずかしい。何より、美月に対して抱いている感情が、以前とは違うものになっていることに、僕自身が戸惑っているのだ。

「分かりました。ぜひお願いします」

「じゃあ、放課後に図書館で待ってる」

「はい」

　チャイムが鳴って、午後の授業が始まる時間になった。

「それじゃあ、また後で」

「はい。ありがとうございます」

　美月は深くお辞儀をしてから、教室に戻っていった。その後ろ姿を見送りながら、僕は自分の胸の高鳴りを感じていた。

　放課後、僕は約束通り図書館で美月を待っていた。プログラミングの入門書を何冊か借りてきて、テーブルに並べている。それぞれの本の特徴を思い出しながら、美月にとって最も分かりやすい説明の順序を考えていた。

「お待たせしました」

　美月が図書館に入ってきた。制服のブレザーを脱いで、白いブラウス姿だ。春の陽射しを受けて、その姿がとても美しく見えた。

「お疲れ様。早速始めよっか」

「はい」

　美月は僕の隣に座った。昨日と同じように、ほのかに桜の香りがする。桜月堂の和菓子の匂いが移ったのだろうか。

「まず、プログラミングって何だと思う？」

「えーと、コンピュータに命令を書くことでしょうか？」

「そうそう。でも、もう少し詳しく言うと、コンピュータに何をしてほしいかを、コンピュータが理解できる言葉で説明することなんだ」

　僕は手元のノートに簡単な図を描きながら説明した。人間とコンピュータの間の意思疎通の仕組みを、できるだけ視覚的に表現する。

「コンピュータが理解できる言葉？」

「プログラミング言語っていうんだ。人間が英語や日本語を話すように、コンピュータにはプログラミング言語で話しかける」

「なるほど」

　美月は真剣に聞いてくれている。その集中した表情を見ていると、なんだか教えている僕も緊張してくる。

「例えば、『Hello, World!』って画面に表示したいとする。それをコンピュータにお願いするには」

　僕はノートに簡単なプログラムを書いた。

```python
print("Hello, World!")
```

「これだけ？」

「これだけ。シンプルでしょ？」

「思ったより簡単なんですね」

「基本はね。でも、複雑なことをしようとすると、どんどん難しくなる」

　僕は別のページに、もう少し複雑なプログラムを書いて見せた。フィボナッチ数列を計算するアルゴリズムだ。

```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

for i in range(10):
    print(fibonacci(i))
```

「わあ、急に難しくなりました」

　美月は苦笑いを浮かべた。

「でも、基本は同じなんだ。コンピュータにお願いしたいことを、順番に書いていくだけ」

「順番に？」

「そう。料理のレシピみたいなもの。まず材料を用意して、次に切って、炒めて、味付けして」

「あ、分かりやすいです」

　美月の表情が明るくなった。

「プログラミングも同じ。まずデータを用意して、次に計算して、結果を表示する。このフィボナッチ数列も、実は数学の規則を順番に実行しているだけなんだ」

「なるほど。レシピですね」

「そういうこと。美月は理解が早いね」

「拓海先輩の説明が上手だからです」

　美月にそう言われて、僕は少し照れた。

　一時間ほど、プログラミングの基本について説明した。アルゴリズムの概念、データ構造の種類、実際のコードの書き方。美月は分からないところがあると積極的に質問してくれるし、理解すると嬉しそうな顔をする。教えている僕の方も楽しくなってきた。

「実は、僕が夢中になっているのは競技プログラミングなんだ」

「競技プログラミング？」

「与えられた問題を、プログラムを書いて解く競技なんだ。例えば、最短経路を求める問題とか、効率的にデータを並び替える問題とか」

「数学の問題みたいですね」

「そうなんだ。数学、論理的思考、プログラミング技術、全てが必要になる。僕は去年、国際情報オリンピックで銅メダルを取ったんだ」

「国際情報オリンピック？」

「世界中の高校生が参加する、プログラミングの大会なんだ。今年もまた出場する予定で」

　美月の目が輝いた。

「すごいです。世界大会なんて」

「でも、まだまだ上には上がいる。アルゴリズムの美しさとか、効率的な解法を見つけた時の感動とか、それが僕をプログラミングに引き込んでいるんだ」

「今日はありがとうございました。とてもよく分かりました」

「どういたしまして。水曜日も楽しんでもらえそう？」

「はい。とても楽しみです」

　美月は目を輝かせて答えてくれた。

「それじゃあ、帰ろうか」

「はい」

　僕たちは本を片付けて、図書館を出た。夕方の校舎は静かで、窓から差し込む夕日が廊下を照らしている。

「拓海先輩は、どうしてプログラミングを始めたんですか？」

　歩きながら、美月が聞いてきた。

「小学校の時に、パソコンでゲームを作ってみたいと思ったのがきっかけかな。最初はScratchっていう教育用のプログラミング言語だったけど」

「Scratch？」

「ブロックを組み合わせてプログラムを作る言語なんだ。子供でも簡単に使えるように設計されている。最初はすごく簡単なものだったけどね。ボールが画面の中で跳ね回るだけのやつ」

「それでも、自分で作ったゲームって特別な感じがしそうです」

「そうなんだ。自分で作ったものが動くって、すごく嬉しいんだよ。それから、どんどん複雑なものを作りたくなって、PythonやC++を覚えていった」

　美月は興味深そうに聞いてくれる。

「それで、どんどんのめり込んでいったんだ。難しい問題を解けた時の達成感とか、新しいアルゴリズムを覚えた時のわくわく感とか。特に、動的プログラミングっていう手法を理解した時は、世界が広がったような気がしたよ」

「拓海先輩が楽しそうに話すから、私も興味が湧いてきました」

「本当に？」

「はい。水曜日が待ち遠しいです」

　学校を出ると、中京区の街並みが夕日に染まっていた。烏丸通りを歩いていると、桜が少し咲き進んでいるのが見える。まだ三分咲きくらいだけれど、だいぶ華やかになっている。

「桜、きれいですね」

「そうだね。今週末には見頃になるかも」

「楽しみです。そういえば、お花見の件はどうしましょうか？」

　ああ、そうだった。先日、美月と一緒にお花見をしようという話をしていた。

「今度の土曜日はどう？天気も良さそうだし」

「いいですね。どこに行きましょうか？」

「学校の桜並木でもいいし、近くの公園でもいい。丸山公園とか」

「学校の桜並木がいいです。思い出の場所ですから」

　美月がそう言った時、僕は少しドキッとした。思い出の場所、か。確かに、僕たちは小学校の頃からあの桜並木を通って登校してきた。でも、今の美月と過ごす時間は、幼い頃とは明らかに違う感情を僕に与えている。

「じゃあ、土曜日の午後に学校で待ち合わせる？」

「はい。お弁当は私が作ってきます」

「え、いいよ。僕も何か持参するから」

「いえいえ、私にお任せください。お礼もかねて」

　美月は嬉しそうに言った。

　桜月堂の前に着くと、美月はいつものように振り返った。店の暖簾には「桜月堂」の文字が風になびいている。

「今日は本当にありがとうございました。水曜日、頑張ります」

「頑張らなくても、楽しんでくれればいいから」

「はい。楽しみにしてます」

　美月は手を振って、店の中に入っていった。

　僕は自分の家に戻りながら、今日の出来事を振り返っていた。美月がプログラミングに興味を持ってくれて嬉しい。それに、土曜日のお花見も楽しみだ。

　最近、美月と過ごす時間が増えている。それが僕にはとても嬉しかった。幼馴染として当然のことなのかもしれないけれど、なんだか特別な感じがする。美月の真剣な表情、理解した時の嬉しそうな顔、ほのかに香る桜の匂い。全てが僕の心を揺さぶる。

　部屋に戻って、僕は水曜日の体験会の準備を続けた。美月に楽しんでもらえるように、しっかりと準備をしよう。今日教えた基本的な内容を踏まえて、どんなプログラムを作ってもらおうか。

　窓の外では、美月の家の明かりが温かく灯っていた。きっと美月も、今日学んだことを思い返しているのだろう。そう思うと、なんだか幸せな気持ちになった。
