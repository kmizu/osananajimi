# 第6話　一緒の帰り道

　月曜日の放課後、僕は美月のクラスの前で待っていた。手にはプログラミング部で使っているThinkPadのノートパソコンが入ったバッグ。今日の部会では、来月のIOI国内予選に向けたグラフアルゴリズムの勉強会をしていた。

　ダイクストラ法の実装を後輩に教えていたとき、ふと美月の顔が浮かんだ。彼女なら、最短経路問題をどんな風に理解するだろう。駅から駅への乗り換えに例えたら、きっとすぐに理解してくれるに違いない。

「拓海先輩、お待たせしました」

　美月が友達と軽く挨拶を交わしてから、僕の方に歩いてきた。制服のリボンが少し曲がっていて、それを直したくなる衝動を抑える。

「お疲れ様。今日はプログラミング部の日じゃないの？」

「今日は部員会議とアルゴリズムの勉強会だけだったから、早めに終わったんだ」

　実際は、美月と一緒に帰りたくて、最後の実装演習を「自主練習」ということにして抜けてきたのだが、それは言わなかった。山田先輩は僕の気持ちを察してか、「藤原は今日はもういいよ」とにやりと笑って送り出してくれた。

「そうなんですね。今日は何を勉強されたんですか？」

　美月は嬉しそうに答えた。その笑顔を見ると、部活を早退した罪悪感が少し和らぐ。

「グラフ理論の最短経路問題。ダイクストラ法っていうアルゴリズムなんだけど」

「難しそうですね」

「いや、実は日常生活でも使ってるんだよ。例えば、GoogleマップとかYahoo!路線情報とか」

　昇降口で靴を履き替えていると、美月の友達が声をかけてきた。

「美月ちゃん、一緒に帰らない？」

「ごめん、今日は先輩と約束があるの」

「あ、そうなんだ。また明日ね」

　友達は僕の方をちらりと見て、意味ありげに微笑んだ。美月の頬がほんのり赤くなったのを、僕は見逃さなかった。

　学校を出ると、夕方の風が心地よかった。桜はもう散り始めていて、歩道に花びらが舞っている。美月と並んで歩いていると、なぜか心拍数が上がる。これも一種のアルゴリズムなのだろうか。if (美月と一緒) then (心拍数++)みたいな。

「土曜日のお花見、楽しかったです」

「僕も楽しかった。美月のお弁当、本当に美味しかったよ。特に出汁巻き卵が絶品だった」

「ありがとうございます。実は、お母さんに教わって何度も練習したんです」

　美月は嬉しそうに笑った。その笑顔を見ると、胸の奥が温かくなる。これもきっと、何かの関数で表現できるのだろうか。いや、そんな単純なものじゃない気がする。

「そういえば、文芸部の朗読会はいつだっけ？」

「今度の金曜日の放課後です。本当に来てくださるんですか？」

「もちろん。楽しみにしてるよ。何を朗読するの？」

「太宰治の『女生徒』です。でも、緊張します」

「大丈夫。美月なら上手にできるよ。練習の時みたいに、自然体でやればいい」

　僕がそう言うと、美月は少し安心したような顔をした。その表情の変化を見逃さないように、僕は美月の顔を見つめていた。最近、美月の表情を読み取るのが楽しくなってきた。これは、パターン認識の一種なのかもしれない。

　いつもの道を歩きながら、僕たちは他愛ない会話を続けた。中京区の住宅街は、この時間になると静かで落ち着いている。時折、自転車で通り過ぎる主婦の姿や、犬の散歩をしている人とすれ違う。

「今日の数学、やっぱり難しかったです」

「どこが分からなかった？」

「三角関数の合成のところです。なんで突然sinとcosが一つの式にまとまるのか」

「ああ、それはね、ベクトルの内積で考えると分かりやすいよ」

　僕は手でジェスチャーを交えながら説明した。美月は真剣な顔で聞いてくれる。

「あ、そうそう」

　美月が思い出したように言った。

「プログラミング部の件、考えてみたんです」

「本当に？」

「はい。兼部できるなら、挑戦してみたいと思って。それに、IOIのことも気になります」

　美月のその言葉で、僕の心が躍った。胸の中で、幸福度を表す変数が最大値を更新した気がした。

「それは嬉しいな。部長の山田先輩も喜ぶと思う。実は今日も『藤原の幼馴染が入部するかも』って話したら、すごく期待してた」

「でも、本当に大丈夫でしょうか？私、まだ初心者ですし、皆さんのレベルについていけるか」

「大丈夫だよ。みんな最初は初心者だったから。僕だって、最初はHello Worldを表示するのに30分かかったんだ」

「本当ですか？」

「うん。セミコロンを忘れてエラーが出て、何が間違ってるのか分からなくて」

「そうですね。それに、拓海先輩が教えてくださるなら」

　美月がそう言った時、僕は胸がドキッとした。美月が僕を頼りにしてくれている。その信頼に応えたい。

「任せて。最初はPythonから始めよう。文法が分かりやすいし、すぐに結果が見えて楽しいから」

「Pythonですか。蛇の名前ですよね？」

「そう。でもプログラミング言語の方は、イギリスのコメディ番組から名前を取ったんだって」

「へー、面白いですね」

　寺町通りの商店街に差し掛かると、桜月堂の看板が見えてきた。美月の家の和菓子店だ。創業50年以上の老舗で、地元の人に愛されている。僕も小さい頃から、ここの練り切りが大好きだった。

「ちょっと寄っていきませんか？」

　美月が提案してくれた。

「いいの？」

「はい。お母さんも拓海先輩にお会いしたがってましたし。それに、今日は新作の試作品があるんです」

「それなら、お邪魔させてもらおうかな」

　正直に言うと、美月ともっと一緒にいたかった。家に帰るまでの時間を少しでも延ばせるなら、それに越したことはない。

　桜月堂の店内は、餡の甘い香りに包まれていた。ショーケースには色とりどりの和菓子が並んでいる。桜餅、花見団子、そして季節の練り切り。どれも職人の技が光る美しい仕上がりだ。プログラミングで言えば、これらは全て完璧にリファクタリングされたコードのようなものだ。無駄がなく、機能的で、そして美しい。

「いらっしゃいませ」

　奥から美月のお母さんが出てきた。美月に似た優しい顔立ちの女性だ。エプロンには少し粉がついていて、今まで作業をしていたことが分かる。

「拓海くん、久しぶりね。相変わらず背が高くなったわね」

「こんにちは、おばさん。お久しぶりです」

「美月がいつもお世話になってます。プログラミングのことも教えてもらってるって聞いてますよ」

「そんな、こちらこそ。美月さんは理解が早くて、教えがいがあります」

　お母さんは温かく迎えてくれた。小さい頃から知っている僕を、今でも変わらず受け入れてくれる。

「今日は新作の桜餅があるの。道明寺粉の配合を変えて、もっちりとした食感にしてみたのよ。よかったら食べていって」

「ありがとうございます」

　僕たちは店の奥の小さなテーブルに案内された。ここは常連客用のスペースで、落ち着いた雰囲気がある。美月のお母さんが淹れてくれた宇治茶と一緒に、桜餅をいただく。

「美味しいです。確かに食感が違いますね。もちもちしてて、でも重くない」

「ありがとう。実は美月も手伝ってくれたのよ」

「え、美月が？」

「桜の葉の塩漬けを巻くのを手伝ってもらいました。葉の向きを揃えて、均等に巻くのが難しいんです」

　美月は少し恥ずかしそうに答えた。その仕草を見ていると、なぜか心が温かくなる。これもアルゴリズムでは説明できない感情だ。

「上手だね。職人技術も、実はアルゴリズムと似てるところがあるんだ。手順を最適化して、効率よく美しいものを作る」

「そんな風に考えたことなかったです。でも、確かに似てるかもしれません」

「そういえば、美月はプログラミングにも興味を持ったみたいね」

　お母さんが言った。

「はい。拓海先輩が楽しそうに話してくれるので。IOIの話とか、アルゴリズムの美しさとか」

「拓海くんは昔から頭がよかったものね。小学生の時から、うちに来ては難しい本を読んでいたわ」

　お母さんにそう言われて、僕は少し照れた。

「美月の方が理解力がありますよ。教えたことをすぐに応用できるんです」

「お互いを褒め合って、仲がいいのね。小さい頃と変わらないわ」

　お母さんは優しく微笑んでいた。その笑顔が、美月とそっくりだった。

　桜餅を食べ終わって、僕たちは店を出た。お母さんは、お土産にと練り切りを包んでくれた。

「お母さん、ありがとうございました」

「また遊びに来てね、拓海くん。今度はプログラミングの話も聞かせて」

「はい、ぜひ」

　美月と一緒に店の外に出ると、夕日が美しく街を照らしていた。寺町通りの古い町並みが、オレンジ色に染まっている。この景色を、もし配列で表現するなら、RGB値の動的な変化を時間軸で記録することになるだろう。でも、美月と一緒に見るこの景色の美しさは、どんなデータ構造でも表現できない。

「お母さん、拓海先輩のこと気に入ってるみたいです」

「そうかな？」

「はい。昔から拓海先輩のことを褒めてましたから。『あの子は将来きっと大物になる』って」

　美月がそう言ってくれて、僕は嬉しかった。でも同時に、少しプレッシャーも感じる。

「僕も、美月の家族のことは昔から好きだよ。温かくて、いつも迎え入れてくれて」

「ありがとうございます。拓海先輩も、もう家族みたいなものですから」

　家族みたい、か。その言葉に、僕は複雑な気持ちになった。嬉しいような、でも何か違うような。

　美月の家の前まで来ると、僕たちは立ち止まった。もう少し一緒にいたいという気持ちを、どう表現したらいいのか分からない。

「今日もありがとうございました。プログラミング部のこと、色々聞けて楽しかったです」

「こちらこそ。桜餅、ご馳走になっちゃって。練り切りももらっちゃったし」

「お母さんも喜んでました。拓海先輩が来てくれて」

　美月は嬉しそうに答えた。夕日に照らされた美月の髪が、きらきらと輝いている。

「明日は、プログラミング部の見学に来る？」

「はい、ぜひ。何時頃がいいですか？」

「放課後すぐでいいよ。コンピュータ室で待ってる」

「分かりました。それじゃあ、また明日」

「はい。また明日」

　美月が家に入っていくのを見送ってから、僕は自分の家に向かった。玄関のドアが閉まる音が、なぜか寂しく聞こえた。

　今日も美月と一緒に過ごせて楽しかった。最近、美月と過ごす時間が増えているけれど、それでもまだ足りないような気がする。これは一体どういうアルゴリズムなんだろう。満足度が上がるほど、要求値も上がるような。

　美月がプログラミング部に入ってくれれば、もっと一緒にいられる。でも、それは僕の都合であって、美月が本当にやりたいことを優先すべきだ。強制したり、誘導したりしてはいけない。

　部屋に戻って、僕はThinkPadを開いた。明日の見学に備えて、初心者向けの課題を準備しよう。まずは、簡単な計算プログラムから。でも、美月なら、もう少し面白いものの方がいいかもしれない。

　そうだ、タートルグラフィックスはどうだろう。視覚的に結果が見えて、プログラミングの楽しさが伝わりやすい。美月が好きそうな、花の模様を描くプログラムを作ってみよう。

```python
import turtle

def draw_petal():
    turtle.circle(50, 60)
    turtle.left(120)
    turtle.circle(50, 60)
    turtle.left(120)

def draw_flower():
    for _ in range(6):
        draw_petal()
        turtle.right(60)
```

　コードを書きながら、美月の反応を想像する。きっと「わあ、きれい」って言ってくれるだろう。そして、「私にも作れますか？」って聞いてくれたら。

　窓の外を見ると、美月の家の明かりが見えた。二階の窓、あれが美月の部屋だ。きっと今頃、宿題をしているのだろう。数学で分からないところがあったら、また教えてあげられる。

　明日も、美月と一緒に帰れるかな。いや、明日は部活見学があるから、もっと長く一緒にいられる。その事実に、僕の心は踊った。

　練り切りの箱を見つめながら、僕は明日のことを考え続けた。美月にプログラミングの楽しさを伝えたい。でもそれ以上に、美月ともっと時間を共有したい。この気持ちは、もう友達以上のものなのかもしれない。

　そんなことを考えながら、僕は机に向かった。画面には、花のプログラムが美しい軌跡を描いていた。
