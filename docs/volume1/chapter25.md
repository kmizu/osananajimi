# 第25話　決意

　日曜日の夜七時、僕は美月の家の前に立っていた。心臓がドキドキして、手が震えている。慎也のアドバイスを思い出し、深呼吸を繰り返した。

　もう逃げるわけにはいかない。IOI準備で美月を後回しにしてしまった罪悪感、高橋先輩への不安、そして何より、美月への想い。すべてに決着をつける時が来た。

　インターホンを押した。

「はい」

　美月のお母さんの優しい声が聞こえてきた。

「こんばんは。拓海です」

「あら、拓海くん。お久しぶりね。どうしたの？」

「美月はいますか？少しお話ししたいことがあって」

「いるわよ。ちょっと待ってて」

　しばらくして、美月が玄関に現れた。私服姿の彼女は、学校で見るのとはまた違った印象だった。でも、その表情には明らかに困惑の色があった。

「拓海先輩、どうしたんですか？こんな時間に」

「話があるんだ。少し時間もらえる？」

　美月は少し躊躇してから答えた。

「分かりました。でも、家の中はちょっと」

「外でも構わない。家の前の公園でどうかな」

「はい」

　僕たちは歩いて近くの小さな公園に向かった。子供の頃、よく一緒に遊んだ場所だ。ブランコや砂場がある、懐かしい公園。夜のライトが優しく二人を照らしている。

　ベンチに並んで座った。美月との距離が近くて、緊張が高まる。

「何のお話ですか？」

　美月が静かに聞いてきた。その声には、以前のような親しみやすさがない。

「まず、謝りたいことがあるんだ」

「謝る？」

「この一ヶ月、IOI準備に夢中になって、美月のことを全然気にかけてなかった。本当にごめん」

　僕は頭を下げた。

「それは、仕方ないことです。拓海先輩の夢ですから」

「いや、仕方なくない。美月は僕にとって大切な人なのに、優先順位を間違えた」

　美月は少し驚いたような顔をした。

「大切な人？」

「うん。でも僕は、自分の目標ばかり追いかけて、美月の気持ちを考えなかった」

「そんなことないです」

「いや、ある。最近の美月を見てれば分かる。明らかに僕を避けてるでしょ？」

　美月は答えに困ったような表情をした。

「それは」

「理由があるなら教えて。僕が何か悪いことをしたなら、ちゃんと謝りたい」

　美月はしばらく黙っていた。そして、小さな声で答えた。

「高橋先輩に言われたんです」

「高橋先輩に？」

「はい。『藤原君は将来有望だから、今は勉強に集中させてあげた方がいい』って」

　僕は驚いた。やはり高橋先輩が関係していたのか。

「それで？」

「『君が甘えさせていると、彼の足を引っ張ることになる。本当に彼のことを思うなら、距離を置くべきだ』とも言われました」

　美月の声が震えていた。その言葉に、僕の胸が痛んだ。

「そんなこと」

「私、拓海先輩の邪魔をしてるのかなって思って」

　美月の目に涙がにじんでいた。

「美月」

「だから、距離を置いた方がいいのかなって。でも、それがこんなに辛いなんて思わなくて」

　僕は美月の手を取った。

「そんなことないよ。美月は僕の邪魔なんかじゃない」

「でも」

「むしろ、美月がいるから頑張れるんだ。美月の存在が、僕の原動力になってる」

　美月は僕を見つめた。

「本当ですか？」

「本当だよ。この一ヶ月、美月がいない日々がどれほど寂しかったか」

「私もです」

　美月が小さく答えた。

「本当？」

「はい。拓海先輩と話せなくて、一緒にいられなくて、とても寂しかったです」

　美月の素直な気持ちを聞いて、僕は嬉しくなった。

「美月、実は僕には、君に伝えたいことがあるんだ」

　僕は意を決して話し始めた。

「最近気づいたんだけど、僕は美月のことを、ただの幼馴染以上に思ってる」

　美月の目が少し大きくなった。

「それは」

「好きなんだ。恋愛感情として、美月のことが好きだ」

　ついに言ってしまった。人生で初めての告白。心臓が破裂しそうなほどドキドキしている。

　美月は驚いた顔をしていたが、嫌そうな表情ではなかった。

「拓海先輩」

「もし迷惑だったら謝る。でも、この気持ちは本当なんだ。美月がいない生活なんて、考えられない」

「迷惑じゃないです」

　美月が小さく答えた。

「本当に？」

「はい。実は、私も」

　美月は恥ずかしそうに俯いた。

「私も、拓海先輩のことを、そういう風に思ってました」

　その言葉で、僕の心が軽くなった。まるで重い荷物を下ろしたような感覚だった。

「本当に？」

「はい。でも、高橋先輩に言われたことが気になって、素直になれませんでした」

「僕もだよ。君の気持ちが分からなくて、不安だった」

　僕たちは微笑み合った。

「それじゃあ」

「うん」

「僕たち、お付き合いするということ？」

　美月が恥ずかしそうに聞いた。

「そうなるのかな」

「はい。よろしくお願いします」

　美月は深々とお辞儀をした。そのしぐさが、とても可愛らしかった。

「こちらこそ、よろしく」

　僕も答えた。

　夜空には星が輝いていて、とても美しかった。京都の街の明かりと星空が、この特別な夜を演出してくれている。

「これからも、一緒にいてくれる？」

「はい。ずっと一緒にいたいです」

　美月の言葉で、僕は心から幸せを感じた。

　長い間抱えていた気持ちが、ようやく美月に伝わった。そして、美月も同じ気持ちでいてくれた。高橋先輩の言葉に惑わされて、お互いに距離を置いてしまった時間が惜しまれる。

　でも、今はそんなことはどうでもいい。美月と僕は、ついに本当の気持ちを確かめ合うことができた。

「美月、ありがとう」

「こちらこそ、ありがとうございます」

　僕たちは手を繋いで、家に向かった。新しい関係の始まりだった。

　幼馴染から恋人へ。長い時間をかけて育んできた関係が、新しい段階に進んだ。

　家の前に着いたとき、美月が僕を見上げた。

「拓海先輩、IOI の結果が出たら、一番最初に教えてくださいね」

「もちろん。美月にも一緒に喜んでもらいたいから」

「はい。結果がどうであれ、私は拓海先輩のことを誇りに思います」

　美月の言葉が、僕の心に深く響いた。

「僕も、美月のことを心から大切にするよ」

「お互いに、支え合っていきましょうね」

　美月は微笑んで言った。

　その夜、部屋に戻った僕は、窓から美月の家を見つめていた。今度は、親近感と愛おしさに満ちた気持ちで。

　IOI の結果がどうあれ、美月と一緒なら、どんな困難も乗り越えられる気がした。

　慎也のアドバイスに従って、素直に気持ちを伝えてよかった。もしあのまま何もしなければ、本当に美月を失っていたかもしれない。

　高橋先輩の言葉は、確かに一理あった。でも、お互いを支え合う関係なら、足を引っ張るどころか、より大きな力を発揮できるはずだ。

　これからは、IOI も美月も、両方大切にしていこう。優先順位を間違えずに、バランスを取りながら。

　美月という最高のパートナーを得た今、僕の人生に新しい章が始まった。プログラミングの技術だけでなく、人を愛することの素晴らしさも学んでいけるだろう。

　星空を見上げながら、僕は心から感謝していた。美月と出会えたこと、長い時間をかけて関係を築けたこと、そして今夜、お互いの気持ちを確かめ合えたこと。

　人生で最も美しい夜のひとつになった。
