# 第30話　恋の告白

　土曜日の朝、僕は美月と一緒に桜並木を歩いていた。
　恋人同士になって初めての、正式なデートだった。

　九月に入り、京都にも秋の気配が感じられるようになった。
　桜の葉が少しずつ色づき始めて、春とは違った美しさを見せている。

「この場所、私たちにとって特別な場所になりましたね」

　美月が桜の木を見上げながら言った。

「そうだね。春にお花見をして、夏にすれ違いがあって、そして今、恋人として歩いている」

「不思議ですね。同じ場所なのに、気持ちが変わると景色も違って見える」

　美月の言葉に、僕は深く頷いた。

「美月、実は今日、君にどうしても伝えたいことがあるんだ」

「伝えたいこと？」

　僕は美月の手を引いて、桜並木の中央にあるベンチに向かった。
　そこは、春に初めて二人でお花見をした場所だった。

「ここに座ろう」

　僕たちはベンチに並んで座った。
　秋の穏やかな日差しが、僕たちを優しく包んでいた。

「美月、この半年間を振り返ってみると、僕の人生が大きく変わったんだ」

「変わった？」

「君と出会い直して、君の優しさや強さを知って、プログラミングだけじゃない世界があることを教えてもらった」

　僕は美月の目を見つめた。

「そして、何より大切なことを学んだ」

「大切なこと？」

「人生で最も価値あるアルゴリズムは、愛することなんだってことを」

　美月の頬が赤くなった。

「アルゴリズムって、問題を解決するための手順だよね。僕にとって、君を愛することが、人生のすべての問題を解決する最適な手順だったんだ」

「拓海先輩」

「IOIの合格も、将来の夢も、すべて君がいるから意味があるんだ」

　僕は立ち上がって、美月の前に膝をついた。

「桜井美月さん」

　美月の目に涙が滲んできた。

「僕はあなたを愛しています」

　僕の言葉に、美月は手で口を覆った。

「これまで『好き』という言葉は何度も伝えたけれど、今日は『愛している』と言いたかった」

「拓海先輩」

「あなたの笑顔を見ていると、どんなに複雑なアルゴリズムよりも美しいと思う。あなたの優しさに触れると、どんなに高性能なシステムよりも心が温かくなる」

　美月の涙が頬を伝った。

「あなたがいない人生なんて、考えられない。あなたと一緒なら、どんな困難なプログラムでも、どんな難しい人生の課題でも、きっと解決できる」

　僕は美月の手を両手で包んだ。

「美月、僕と一緒に、素晴らしい人生を書いていってくれませんか？」

　しばらくの沈黙の後、美月が答えた。

「藤原拓海先輩」

　美月の声は震えていた。

「私も、拓海先輩を愛しています」

　僕の胸が熱くなった。

「拓海先輩がプログラミングのことを話すとき、目がキラキラ輝いているのを見ていると、私も一緒に夢を追いかけたくなります」

「美月」

「拓海先輩がいると、毎日が特別な日になります。一緒にいるだけで、世界中が明るく見える」

　美月は涙を拭きながら続けた。

「私にとって、拓海先輩との恋愛は、最高のプログラムです。バグのない、完璧で美しいコードです」

　僕は思わず笑った。

「君も技術的な比喩を使うようになったんだね」

「拓海先輩の影響です」

　僕たちは見つめ合って、そして自然にキスをした。
　桜並木の秋風が、僕たちを祝福するように頬を撫でていった。

「美月、ありがとう。君が僕の人生に来てくれて、本当に良かった」

「私こそ、ありがとうございます。拓海先輩に出会えて、人生が変わりました」

　僕たちは抱き合った。
　春の桜、夏の新緑、そして今の秋の色づいた葉。
　この場所で、僕たちは成長し、愛を深めてきた。

「美月、これからもずっと、一緒にいよう」

「はい。ずっと一緒です」

　その時、僕のスマートフォンに通知が来た。
　IOI本戦の詳細な案内だった。

「来月、オーストラリアに出発することになる」

「頑張ってきてください。私も、拓海先輩がいない間、プログラミングの勉強を続けます」

「帰ってきたら、君のコードを見せてもらうのが楽しみだ」

「恥ずかしいですが、頑張ります」

　美月は嬉しそうに微笑んだ。

「そうだ、君に約束してもらいたいことがある」

「約束ですか？」

「僕がIOI本戦で頑張っている間、君も自分の夢を見つけて欲しいんだ」

「私の夢？」

「君にはプログラミングの才能があるけれど、それ以外にも文学や芸術に対する感性がある。君だけの特別な夢を見つけて欲しい」

　美月は真剣に考えた。

「分かりました。拓海先輩がいない間に、私なりの目標を見つけてみます」

「ありがとう。そうすれば、僕が帰ってきた時、お互いに成長した姿を見せ合えるね」

「はい。とても楽しみです」

　午後、僕たちは中京区の商店街を歩いた。
　錦市場で京都の食材を見て回り、新京極で本屋に立ち寄った。

「この本、面白そうですね」

　美月が手に取ったのは、コンピューターサイエンスと文学の関係について書かれた本だった。

「君らしい選択だね。買ってみよう」

「ありがとうございます」

　夕方、桜月堂で美月の両親と一緒にお茶をいただいた。

「IOI本戦、頑張ってきてくださいね」

　お母さんが言ってくれた。

「はい。美月さんのことをよろしくお願いします」

「こちらこそ。拓海くんの夢、家族みんなで応援していますから」

　お父さんも温かく言ってくれた。

　夜、美月と一緒に自宅の前まで帰ってきた。

「今日は本当に楽しかったです」

「僕も。君と過ごす時間は、いつも特別だ」

　僕は美月の手を握った。

「美月、改めて言わせて」

「はい」

「君を愛している。この気持ちは、僕がどこにいても変わらない」

「私も愛しています。どんなに遠く離れても、心はいつも一緒です」

　僕たちは最後にもう一度キスをした。
　京都の夜空に星が輝いて、僕たちの愛を見守ってくれているようだった。

　家に入って、僕は窓から美月の家を見つめた。
　隣同士に住んでいることの幸せを、改めて感じていた。

　スマートフォンに美月からメッセージが届いた。

【今日は本当にありがとうございました。拓海先輩と過ごした時間、一生忘れません。愛しています】

　僕は笑顔で返信した。

【こちらこそありがとう。君がいてくれて、僕は世界一幸せな男だよ。愛してる】

　部屋のデスクに、IOI本戦の準備資料が積まれている。
　でも今は、それよりも美月との愛の方が心を満たしていた。

　プログラミングの世界では、効率的で美しいコードを「エレガント」と呼ぶ。
　僕と美月の恋愛も、まさにエレガントだった。
　
　無駄がなく、美しく、そして完璧に機能している。

　来月はオーストラリアで世界の天才たちと競うことになる。
　でも、美月という最高のパートナーがいる僕には、怖いものはなかった。

　愛という最強のアルゴリズムを手に入れた僕は、どんな困難も乗り越えられる。
　そう確信していた。

　窓の外で、桜並木が静かに風に揺れている。
　春から始まった僕たちの物語は、愛の告白というクライマックスを迎えた。

　でも、これは終わりではない。
　新しい章の始まりなのだ。

　美月と僕の愛の物語は、これからもずっと続いていく。
　二人で書いていく、最高のプログラムとして。

【第1巻　完】
