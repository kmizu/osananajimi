# 第22話　誤解

　金曜日の放課後、僕は図書館の奥の席でIOI過去問と向き合っていた。あと三日でオンライン予選の最終課題提出。今年こそは予選突破を果たし、日本代表選考会への切符を掴みたい。

　画面に表示されているのは、グラフ理論とフローアルゴリズムを組み合わせた応用問題。最小スパニングツリーを構築してから、ネットワークフローで最適化を図る戦略だが、制約条件が厳しく、計算量の削減に苦戦している。実装に三時間かけたが、まだタイムアウトが発生する。

　そんな時、隣の席に誰かが座った。振り返ると、美月が数学の教科書を抱えて微笑んでいた。

「拓海先輩、お疲れ様です。今日も難しい問題と格闘中ですね」

　美月の声には、いつものような優しさがあった。でも今の僕は、グラフのエッジの重みづけで頭がいっぱいで、彼女の気持ちを受け止める余裕がなかった。

「ああ、美月。お疲れ様」

　僕は半ば機械的に返事をした。Union-Find木の経路圧縮の最適化について考えながら。

「選考会の準備、本当に大変そうですね。こんなに複雑な問題ばかり解いていて」

「うん。でもこれくらいできないと、本番で太刀打ちできないから。今はフローアルゴリズムの実装を詰めてるところなんだ」

　僕はキーボードを叩きながら答えた。Ford-Fulkerson法でボトルネック解消を図っているが、まだ改善の余地がある。

「そうですか。拓海先輩の努力を見ていると、私ももっと数学を頑張ろうって思います」

　美月は数学の教科書を開いた。参考書を何冊も持参している。彼女なりに真剣に勉強に取り組んでいるのが分かる。

　でも僕は、再びコードの修正に意識を集中させた。今は一秒でも無駄にできない。

　二時間後、ようやくグラフ問題の最適解を得られた。アルゴリズムの計算量もO((V+E)logV)で実用的。肩の力を抜いて、隣の美月を見る。彼女は静かに数学の問題と取り組んでいる。

「美月、何の問題をやってるの？」

「数学III の積分です。置換積分と部分積分の組み合わせで、ちょっと複雑で」

「難しいの？」

「はい。でも、拓海先輩に以前教えてもらった方法を思い出しながら、一歩ずつ進めています」

　美月は有益そうに答えたが、その声には微かな寂しさが混じっていた。

「分からないことがあったら聞いて。今は時間あるから」

「でも、拓海先輩のIOI の準備の方が大切ですよね。選考会、もうすぐでしょ？」

　美月の言葉に、僕は心臓がギュッと締め付けられるような感覚を覚えた。彼女はいつも僕のことを考えてくれている。でもその優しさが、今の僕には重荷に感じられた。

「いや、美月の勉強も大事だよ。僕は美月のことを、いつでも大切に思ってるから」

　僕がそう言うと、美月は少し驚いたような顔をした。でも、その後で微かに笑った。

「ありがとうございます。でも、やっぱり拓海先輩の夢の方が大事です」

　その時、図書館の入口から生徒会副会長の高橋先輩が入ってきた。三年生で、身長も高く、学年でもトップクラスの成績を誇る優等生だ。僕たちのテーブルを見つけると、直線的に歩いてきた。

「桜井さん、藤原君。勉強お疲れ様です」

「高橋先輩、こんにちは。いつもお世話になっております」

　美月はいつものように礼儀正しく挨拶した。先輩に対する敬語も完璧だ。

「桜井さん、例の件ですが、来週の火曜日にお時間をいただけませんか？生徒会の企画で、いくつか相談したいことがありまして」

「はい、大丈夫です。お役に立てるか分かりませんが」

　僕はその交換を聞きながら、キーボードを叩く手が無意識に止まっていた。美月は先輩の用件を快く引き受けている。生徒会の企画と言っているが、本当にそれだけなのだろうか。

「それでは、火曜日の放課後に生徒会室でお待ちしております」

「はい、よろしくお願いします」

　高橋先輩は僕にも軽く会釈してから去っていった。彼の歩き方には、何か目的を達成したような満足感が漂っていた。

「生徒会の仕事って、具体的には何なの？」

　僕は美月に聞いてみた。グラフ理論のことは一時的に頭から遠のいていた。

「文化祭の企画運営について、いくつかアドバイスをいただく予定です。私たちの部の展示も、とても好評でしたから」

「そうなんだ。でも、なんで高橋先輩が特別に美月に？」

　僕の質問に、美月は少し困ったような表情をした。

「実は、それ以外のことも話したいことがあるとおっしゃっていました」

　美月の言葉に、僕の胸に嫌な予感が広がった。

「それ以外って、何の話？」

「詳しくは分からないのですが、個人的な相談もあるとか」

　美月の答えは曖昧だったが、その表情からは何かを隠しているような印象を受けた。

　その日の夜、僕は部屋の机に向かっていたが、まったく集中できなかった。コードを書いても、基本的なアルゴリズムでさえミスを犯してしまう。頭の中で、高橋先輩と美月の会話がリプレイされ続けている。

「それ以外のことも話したい」って、何のことだろう。生徒会副会長という立場、優秀な成績、面倒見の良い性格。美月が彼に好意を持っても不思議ではない。

　僕はプログラミング以外に、特に誇れるようなものはない。内気だし、コミュニケーション能力も高いとは言えない。リーダーシップもない。美月が僕よりも高橋先輩を選んでも、不自然ではない。

　月曜日の朝、早めに学校に着くと、美月がクラスメイトの女子たちと楽しそうに話しているのが見えた。彼女の笑顔はいつもと変わらない明るさだった。

「おはようございます」

　僕が声をかけると、美月は振り返った。でもその反応は、以前よりもワンテンポ遅いように感じられた。

「おはようございます、拓海先輩」

　美月の声はいつもよりも小さく、どこか他人行儀な感じがした。クラスメイトたちと話している時の明るさとは、明らかに違っている。

「何かあった？最近、なんだか元気ないように見えるけど」

「いえ、特に何も。ただ、最近忙しくて」

　美月の答えは素っ気ないものだったが、その目は僕の顔を直視していなかった。まるで僕と話すことが、負担になっているかのようだった。

　昼休み、僕はいつものように図書館で勉強していた。美月も来るかと密かに期待していたが、結局彼女の姿は見えなかった。代わりに来たのは、一年生の同級生たちだった。美月は彼らと一緒に食堂で昼食を済ませたのだろう。

　放課後、プログラミング部に向かうと、美月の姿はなかった。部員の一人に聞いてみる。

「美月は今日、部活に来てないの？」

「ああ、先に帰ったよ。文芸部の方で何か用事があるって言ってた。最近、こっちにはあまり顔を出さないんだよな」

　慎也の言葉に、僕の心に重い石が沈んでいくようだった。美月の足が、明らかに僕から遠ざかっている。

　その夜、僕はスマートフォンを手に、美月へのメッセージを何度も書き直した。結局、シンプルな内容に落ち着いた。

【今日は部活に来なかったのね。文芸部の方で何かあったの？】

　三十分後、美月から返事が来た。いつもよりも簡潔で、事務的な印象を受けた。

【はい。文芸部の準備がありました。選考会の勉強、頑張ってください。お疲れ様でした。】

　以前なら、「お疲れ様です」の前に絵文字や「！！」がついていたのに、今日の返事はどこか素っ気ない。まるで先生から生徒への連絡のようだった。

　火曜日の放課後、僕は図書館でデータ構造の問題と格闘していたが、まったく集中できなかった。セグメント木の遅延評価を実装しようとしているのに、考えがまとまらない。

　美月は今頃、高橋先輩と一対一で話しているのだろう。生徒会室という密室で、二人きりで。何を話しているのだろうか。本当に生徒会の仕事だけなのだろうか。

　一時間後、美月が図書館に入ってきた。彼女の表情は、いつもよりも沈んでいるように見えた。何か重い話を聞いたのだろうか。

「お疲れ様。高橋先輩との話は終わった？」

　僕が声をかけると、美月は少しビクッとした。まるで不意打ちを受けたかのようだった。

「あ、拓海先輩。お疲れ様です」

「生徒会の企画の話はどうだった？」

「はい。文化祭の運営について、いくつかアドバイスをいただきました」

　美月は簡潔に答えたが、その目は下を向いていた。何かを隠しているように見える。

「そうなんだ。他には何も？」

　僕は反射的に質問していた。美月の様子が明らかにおかしい。

「特に何も。ただの生徒会の仕事です」

　美月は答えたが、その声には確信がなかった。

「美月、最近本当に忙しくて、あまり時間が取れなくてごめん。一緒にいる時間が減っちゃって」

「いえ、大丈夫です。拓海先輩の夢のためですから。私は、それを応援したいです」

　美月の言葉は優しかったが、その声には何か無理をしているような悲しみが混じっていた。

「選考会が終わったら、また一緒に時間を過ごそう。以前みたいに」

「はい。でも、もしかしたらその頃には」

　美月は言いかけて、口をつぐんだ。何か言いたいことがあったようだが、結局何も言わなかった。

　その夜、僕は部屋で一人、美月との会話を何度も思い出していた。彼女の表情、言葉、そして途中で止めた発言。全てが気になって仕方なかった。

　明らかに、僕たちの間に距離ができている。それは僕のIOI準備のせいだけではない。何か別の、もっと根深い理由があるのではないだろうか。

　高橋先輩が関係しているのだろうか。彼が美月に何か言ったのだろうか。生徒会の仕事以外の「それ以外のこと」とは、一体何だったのだろう。

　気になって仕方ないが、今は選考会のことだけを考えるべきだ。人生で一度きりのチャンス。ここで集中できなければ、全てが水の泡になってしまう。

　でも僕は知らなかった。美月が高橋先輩から聞いた言葉に、どれほど深く傷ついているかを。そして、その傷が彼女を僕から遠ざけていることを。
