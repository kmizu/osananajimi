# 第21話　すれ違いの始まり

　文化祭が終わって一週間が過ぎた。
　六月に入り、僕は今年最大の挑戦に向けて、人生を賭けるような覚悟で日々を送っていた。

　国際情報オリンピック（IOI）日本代表選考会まで、残り二ヶ月。
　去年は予選通過できずに悔しい思いをした。今年こそは、絶対に日本代表の座を掴みたい。

「拓海、最近本当に鬼気迫る感じだな」

　プログラミング部で、慎也が僕の画面を覗き込みながら言った。モニターには、最短路問題のDijkstra法の実装が表示されている。

「AtCoderのARC今月分、全部解き直してるんだ。制限時間を本番の半分に設定して」

「半分って、きつくない？」

「本番で余裕を持つためには、これくらいじゃないと」

　僕は手を止めずに答えた。画面には実行時間の測定結果が表示されている。1.2秒。まだ改善の余地がある。

　その時、美月が部室に入ってきた。いつものように明るい笑顔で挨拶してくれるが、僕はセグメント木の実装でバグを発見して、頭の中がアルゴリズムでいっぱいだった。

「お疲れ様です。今日も難しそうな問題ですね」

「あ、美月。お疲れ様」

　僕は画面から目を離さずに答えた。Range Minimum Queryで時間計算量がO(log n)にならない。どこかでミスがある。

　美月は僕の後ろに立って、画面を見ていた。その気配は感じていたが、今はコードの修正に集中しなければならない。

「拓海先輩、今日は一緒に帰りませんか？久しぶりに桜月堂の新作も試してもらいたくて」

　部活終了のチャイムが鳴った後、美月が期待を込めた声で聞いてきた。

「ごめん、図書館で過去問演習の続きをしなくちゃ。今日はJOI2019の第4問が解けなくて」

「そうですか」

　美月の声のトーンが少し下がったが、僕はセグメント木の更新処理を見直すのに夢中だった。

「明日は？京都国立博物館で面白い展示があるって聞いて」

「明日も朝から図書館に行く予定なんだ。競技プログラミングって、一日でも練習を怠ると感覚が鈍るから」

「分かりました。お疲れ様です」

　美月は一人で帰っていった。

　図書館で一人、IOI過去問題集とにらめっこしていると、胸の奥に鈍い痛みを感じた。さっきの美月の表情を思い出す。期待していた顔から、落胆した表情に変わった瞬間を。

　でも、IOI日本代表は僕の夢だ。小学生の時に初めてプログラミングに触れて以来、ずっと憧れ続けてきた舞台。去年の悔しさを忘れることはできない。予選敗退の瞬間、悔し涙を流した自分を。

　今年は絶対に代表になる。そのためには、全てを犠牲にしても構わない。

　翌朝、午前七時。図書館の開館と同時に席を確保した。今日の目標は、グラフ理論の応用問題を5問解くこと。最短路、最小全域木、ネットワークフロー。全て制限時間の7割で解けるようになるまで繰り返す。

「おはようございます」

　三十分後、美月が図書館に入ってきた。僕と同じ早起き組だ。

「おはよう。今日も早いね」

「拓海先輩が毎朝ここにいるって聞いて、私も頑張ろうと思って」

　美月は僕の斜め向かいの席に座った。控えめな距離感が、彼女の気遣いを表している。

「数学の予習？」

「はい。拓海先輩に負けないように、私も勉強頑張りたくて」

　美月が静かに数学の教科書を開く音が、図書館の静寂に溶け込んでいく。こうやって彼女と同じ空間で勉強するのは、文化祭前以来だった。ほんの一ヶ月前なのに、もう遠い昔のことのように感じられる。

　でも、今は感傷に浸っている場合じゃない。手元のノートには、ダイクストラ法の改良版が書かれている。優先度付きキューを使った実装で、時間計算量O((V+E)log V)を達成したい。

　集中しようとしたその時、美月の存在が気になってしまう。彼女の鉛筆の音、ページをめくる音、小さなため息。全てが僕の意識を引きつけて、アルゴリズムの思考を妨げる。

「美月、悪いんだけど、席を変わってもらえる？」

「え？」

　美月は驚いたような顔で僕を見た。

「一人の方が集中できるから。ごめん」

「あ、はい。分かりました」

　美月は教科書とノートをまとめて、三つ隣のテーブルに移った。その時の表情を見てしまった。理解しようとしているけれど、明らかに傷ついている顔だった。

　胸が痛んだが、僕は自分に言い聞かせた。今は集中するべき時だ、と。

　昼休みのチャイムが響いても、僕は図書館に残った。動的プログラミングの問題で行き詰まっている。ナップサック問題の応用だが、制約条件が複雑で最適化が思うようにいかない。

「拓海先輩、お弁当一緒に食べませんか？今日は拓海先輩の好きな卵焼きも作ってきました」

　美月が手作りの弁当箱を持って、そっと声をかけてきた。桜月堂の包装紙で包まれた弁当は、いつものように丁寧に作られているのが分かる。

「ごめん、今日はコンビニのパンで済ませるから。食事時間ももったいなくて」

「そうですか。でしたら、せめてこれだけでも」

　美月は小さな包みを机に置いた。中を見ると、一口サイズの和菓子が入っている。

「ありがとう。後で食べるよ」

　僕は包みを脇に置いて、再びコードに向き合った。美月が一人で弁当を広げる気配を感じながら、メロンパンを齧る。味気ない食事だったが、時間を節約するためには仕方ない。

　美月が作ってくれた和菓子のことは、その時は忘れてしまっていた。

　放課後のプログラミング部でも、僕は選考会の準備を続けていた。今日はセグメント木の実装を完璧にする予定だった。Range Update, Range Queryを効率的に処理するデータ構造。IOIでは必須の技術だ。

「拓海、そのコード、3時間ぐらい書き続けてるだろ？」

　山田先輩が僕の肩を叩いた。

「はい。でも、まだバグが取りきれなくて」

「たまには休憩しないと、逆に効率悪いぞ」

「大丈夫です。今、いいところなんで」

　先輩は少し困ったような顔をして、部室の窓の外を見た。

「そういえば、美月さん、最近元気ないな」

　その言葉で、僕の手が一瞬止まった。

「元気ない？」

「ああ。この前も一人で弁当食べてたし、なんか寂しそうで」

　胸がチクリと痛んだ。でも、すぐにコードの方に意識を戻す。

「選考会が終わったら、きちんと話します」

「本当に大丈夫か？美月さん、お前のことを本当に大切に思ってるぞ。そういう人を失うのは、人生で一番もったいないことだと思うけどな」

　先輩の重い言葉が、僕の心の奥深くに沈んでいった。でも今は、セグメント木の遅延評価の実装に集中しなければならない。

　夜八時、僕は一人で京都の住宅街を歩いていた。美月は六時頃に先に帰ったようだった。普段なら一緒に歩く烏丸通りも、今日は一人だ。

　家に着いて振り返ると、桜月堂の暖簾はもうしまわれていた。美月の部屋の明かりが、二階の窓から漏れている。彼女は今、何をしているのだろう。宿題だろうか、それとも読書だろうか。

　文化祭の時は、毎日のように一緒に準備をして、一緒に帰って、本当に楽しかった。美月の笑顔、真剣な表情、夜店での弾けるような笑い声。全てが昨日のことのように鮮明に思い出せる。

　でも今は、IOIが全てだ。日本代表になるという夢。去年果たせなかった目標。プログラミングに人生を賭けると決めた以上、他のことは後回しにするしかない。

　美月なら理解してくれるはずだ。僕の夢を、誰よりも応援してくれていたのは美月なのだから。

　部屋で机に向かい、深夜2時まで問題演習を続けた。Union-Find木、フローアルゴリズム、文字列処理。一つずつ確実にマスターしていく。

　パソコンの画面に向き合いながら、時々美月のことが頭をよぎったが、その度に意識を引き戻した。今は集中する時だ。

　しかし僕は気づいていなかった。美月がどれほど我慢をして、どれほど寂しい想いを抱えているかを。そして、二人の間に、取り返しのつかない溝ができ始めていることを。

　翌朝、昇降口で美月と会った。

「おはようございます」

「おはよう」

　美月はいつものように明るく挨拶してくれたが、その笑顔は少し作られたもののように見えた。以前の自然な明るさとは、何かが違っていた。

　でも僕は、頭の中でセグメント木の実装方法を考えながら歩いていたので、その微妙な変化に気づくことはできなかった。
