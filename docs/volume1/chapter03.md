# 第3話　プログラミング部の誘い

　金曜日の放課後、僕はプログラミング部の部室で集中していた。画面に映るのは、最近取り組んでいる幾何アルゴリズムのコード。二次元平面上の点群から最小包囲円を求めるWelzlのアルゴリズムの実装だ。

　期待時間計算量O(n)という美しい結果に魅せられて、ここ数日はこの問題ばかり考えていた。ランダム化を利用した分割統治法――理論的には美しいが、実装となると微妙な境界処理に頭を悩まされる。

「拓海、新入部員の勧誘はどうするんだ？」

　部長の山田隆志先輩が声をかけてきた。三年生の山田先輩は来年大学受験を控えているが、部の運営にはとても熱心だ。

「そうですね。今年はどのくらい入ってくれるでしょうか」

　僕は画面から目を離して振り返った。

「去年は二人だったからな。今年はもう少し増えてほしいところだ」

　プログラミング部の現実は厳しい。全校生徒の中で、真剣にコンピュータ科学に興味を持つ人は本当に少ない。多くの人にとって、プログラミングはまだ「難しくて近寄りがたいもの」なのだ。

「拓海のIOI銅メダルの実績があるから、今年は期待できるんじゃないか？」

　副部長の田村先輩がそう言ってくれた。でも僕は複雑な気持ちだった。確かに実績は欲しいけれど、それがプレッシャーにもなっている。

「でも、プログラミングって敷居が高いと思われがちですよね」

「確かに。アルゴリズムとかデータ構造とか、言葉だけで敬遠されそうだ」

　同期の慎也が口を挟んだ。

「そうなんです。でも実際にやってみると、論理的な美しさがあって面白いんですけど」

「その面白さをどう伝えるかが問題だな」

　山田先輩は腕を組んで考え込んだ。

「そういえば、拓海の幼馴染の桜井さん、興味ありそうじゃない？」

　慎也が突然そんなことを言った。

「美月が？」

「この前、プログラミングのことを熱心に聞いてきたって言ってたじゃん」

　そういえば、図書館で数学を教えた時、微分がプログラミングでも使われるという話をしたら、美月はとても興味深そうに聞いてくれた。

「でも、美月は文系だし……」

「文系でもプログラミングはできるよ。最近は機械学習とか、数学的背景が重要になってるし」

　田村先輩が指摘した。

「それに、女子部員がいると部の雰囲気も変わるしな」

　山田先輩も乗り気だ。

「でも、無理に誘うのは……」

「無理にじゃなくて、まずは体験会から始めてみたらどうだ？プログラミングの本質を体験してもらう」

　慎也の提案に、他の部員たちも賛成した。

「今度、体験会を開いてみるか。アルゴリズムの面白さを実感してもらえるような」

「いいですね。ヴィジュアルで分かりやすいものがいいかも」

「ソートアルゴリズムの可視化とか？バブルソートとクイックソートの違いを見せるとか」

「フラクタル図形の生成なんかも面白そうだ」

　みんなでアイデアを出し合っているうちに、体験会の企画がまとまってきた。

　部活が終わって、僕は一人で部室を片付けていた。美月をプログラミング部に誘うなんて、今まで考えたこともなかった。でも、確かに美月は僕の話を興味深そうに聞いてくれる。

　数学の美しさに感動する美月なら、アルゴリズムの美しさも理解してくれるかもしれない。

「拓海、まだいたのか」

　振り返ると、慎也が戻ってきていた。

「片付けしてました」

「桜井さんのこと、どうするんだ？」

「どうするって……」

「体験会に誘うかどうかだよ」

　慎也は真剣な顔をしている。

「美月が興味を持つかどうか分からないし」

「聞いてみなきゃ分からないじゃん。それに、お前と一緒にいる時間が増えるんだぞ？」

　その言葉に、僕の頬が熱くなった。

「そんなこと考えてないよ」

「嘘つけ。お前、桜井さんのこと好きだろ？」

　慎也の直球な質問に、僕は返答に困った。好きかどうかなんて、まだ自分でもよく分からない。でも、美月と一緒にいると心が落ち着くし、もっと彼女のことを知りたいと思う。

「とにかく、一度聞いてみろよ。きっと喜ぶって」

「そうかな……」

「絶対そうだって。桜井さん、お前のこと尊敬してるじゃん」

　慎也は自信満々に言った。

　学校を出ると、ちょうど美月が昇降口から出てくるところだった。一年生の部活も終わったばかりなのだろう。

「拓海先輩、お疲れ様です」

「美月も、お疲れ様」

「今日はプログラミング部だったんですか？」

「そう。美月は何部に入ったの？」

「文芸部です。小説を読むのが好きなので」

　文芸部か。美月らしい選択だと思う。

「楽しい？」

「はい。皆さん優しくて、読書会とかもあって」

　美月は嬉しそうに話してくれた。

　一緒に歩きながら、僕は体験会のことを話そうかどうか迷っていた。でも、慎也の言葉を思い出して、勇気を出してみることにした。

「美月、今度プログラミング部で体験会をやるんだ」

「体験会ですか？」

「うん。プログラミングやアルゴリズムがどんなものか、実際に体験してもらう会。ソートの仕組みを可視化したり、簡単なゲームを作ったり」

「面白そうですね！」

　美月は目を輝かせた。

「数学が実際にどうプログラムに応用されるか、体験できるよ。この前話した微分も、実はグラフィックスで使われてたりする」

「本当ですか？もしよかったら、見学に来てもいいでしょうか？」

「もちろん！他の部員もみんな歓迎するって言ってた」

「ありがとうございます。ぜひ参加させてください」

　美月の素直な反応に、僕は胸が躍った。思ったよりも簡単に承諾してもらえて、むしろ少し拍子抜けした。

「本当に？無理しなくても……」

「無理じゃないです。前から、拓海先輩がどんなことをしているのか興味があったんです」

「そうなんだ」

「プログラミングって、とても難しそうなイメージがあるんですけど、拓海先輩が楽しそうに話してくれるので、きっと面白いんだろうなって」

　美月のその言葉で、僕の胸が温かくなった。美月が僕の話を聞いて、興味を持ってくれていたなんて。

「ありがとう。きっと楽しんでもらえると思う。アルゴリズムの美しさを、美月にも感じてもらえたら」

「楽しみです」

　桜月堂の前まで来ると、美月はいつものように振り返った。

「体験会はいつですか？」

「来週の水曜日の放課後。詳しい時間はまた連絡するね」

「はい。ありがとうございます」

　美月は嬉しそうに手を振って、店の中に入っていった。

　僕は自分の家に戻りながら、来週の体験会のことを考えていた。美月にプログラミングの本質――論理的思考の美しさ、アルゴリズムの優雅さを伝えられるだろうか。

　もし美月が興味を持ってくれたら、一緒に部活動ができる。同じ目標に向かって、一緒にコードを書いたり、アルゴリズムについて議論したり……

　でも、それ以上に、美月が僕の話に興味を持ってくれていたことが嬉しかった。幼馴染として見てくれているだけじゃなくて、僕という人間に興味を持ってくれている。

　部屋に戻って、僕は体験会の準備を始めた。美月にも分かりやすいように、できるだけ視覚的で楽しいプログラムを考えよう。

　まずはソートアルゴリズムの可視化から始めて、それから簡単なゲーム制作に移る。美月が数学に興味があることを考えると、フラクタル図形の生成なんかも面白がってくれるかもしれない。

　窓の外を見ると、美月の家の明かりが見えた。きっと今頃、宿題か読書をしているのだろう。

　来週の水曜日が楽しみだった。美月がプログラミングの魅力を理解してくれるといいな。そして――できれば、僕ともっと多くの時間を過ごしてくれるといいな。

　そんなことを思いながら、僕は体験会用のプログラムのコーディングを始めた。
